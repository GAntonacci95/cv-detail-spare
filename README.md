# cv-detail-spare

## EN 🇬🇧

### DocShredder (2024)

A creative and draft project to censor
personal information from documents.

To do it, documents are transformed into
images. From such images, sections are
extracted using python and opencv2 to
emulate an OCR behavior. Then a borders
deletion, segmentation, windowing, image to
text, and personal information recognition
strategy is applied to perform the
censorship.

Unfortunately, the resulting algorithm
computational cost is excessive.

Hard skill: opencv2, openly available vision and nlp models;

Soft skill: complexity, scenario
analysis, information handling.

## IT 🇮🇹

### DocShredder (2024)

Bozza di progetto creativo per censurare informazioni personali da documenti.

Per farlo, i documenti sono trasformati in immagini. Da queste, delle sezioni sono estratte utilizzando python ed opencv2 per emulare il comportamento di un OCR. Poi, una strategia di rimozione bordi, segmentazione, finestramento, conversione immagine a testo e riconoscimento di informazioni personali viene applicata per effettuare la censura.

Sfortunatamente, il costo computazionale dell'algoritmo risultante è eccessivo.

Skill verticali: opencv2, modelli vision ed nlp liberamente disponibili;

Skill orizzontali: complessità, analisi scenario, gestione informativa.
